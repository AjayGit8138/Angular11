Create a Angular-App

  • To Install a Angular-Version11 Globally by Using command - "npm install -g @angular/cli@11.0"
  • To create a New angular app in specified Directory using command - "ng new project name"
  • After succesfull creation of angular project,Goto your specified repositary and open with any angular supported text editors,currently by using Visual studio code.
  • To Run a angular Application in Browser using by this Command- "ng serve --open" will open with specified default Browser

Install Additional Design Requirements

  • To Install Bootstrap Design Libarary To our Project Usin this command - "npm install bootstrap --save" after install the packages is added into a node_modules
  • To Install Angular Internal specified Material Design by using this command - "ng add @angular/material"

Angular structural CLI commands

  • Create Component: To use "ng g component 'component Name' "
  • Create a service: To use "ng g service 'service Name'"
  • Create a pipe:To use "ng g pipe 'pipe-name'"
